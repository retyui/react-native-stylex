# i18n ðŸ—º

We use [I18nManager](https://github.com/facebook/react-native/blob/e36247030575e93c1d7087cdb04e1498e9577056/Libraries/ReactNative/I18nManager.js) to handler correct direction

**!!! âš ï¸ React Native include special mechanism to revert 'left' to 'right' value [`I18nManager.swapLeftAndRightInRTL(true)`](https://reactnative.dev/blog/2017/11/06/react-native-monthly-5#facebook) âš ï¸ !!!**

- `i18n({ ltr, ltr })`
- `rtl(value)`
- `ltr(value)`

It means that don't need to use this apis above for setting next values:

- Position:
  - (left|right) â†’ (start|end)
- Margin:
  - margin(Left|Right) â†’ margin(Start|End)
- Padding:
  - padding(Left|Right) â†’ padding(Start|End)
- Border:
  - borderTop(Left|Right)Radius â†’ borderTop(Start|End)Radius
  - borderBottom(Left|Right)Radius â†’ borderBottom(Start|End)Radius
  - border(Left|Right)Width â†’ border(Start|End)Width
  - border(Left|Right)Color â†’ border(Start|End)Color

```js
import { i18n } from "react-native-stylex/i18n";

// âœ… Good https://reactnative.dev/blog/2017/11/06/react-native-monthly-5#facebook
good: {
   start: 20,
}

// âŒ Bad
bad: {
   ...i18n({ ltr: { left: 20 }, rtl: { right: 20 } }),
}
```

**Example:**

```js
import { makeUseStyles } from "react-native-stylex";
import { i18n, rtl, ltr } from "react-native-stylex/i18n";

export const useStyles = makeUseStyles(() => ({
  root: {
    // you can pass styles
    ...i18n({ ltr: { left: 20 }, rtl: { right: 20 } }),

    // or string value for property
    textAlign: i18n({ ltr: "left", rtl: "right" }),
  },
  cell: {
    // styles variant
    ...ltr({ left: 0 }),
    // specific value variant
    alignItems: ltr("flex-start"),
  },

  row: {
    // styles variant
    ...rtl({ right: 0 }),
    // specific value variant
    alignItems: rtl("flex-end"),
  },
}));
```
